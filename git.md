## Git - VERSION CONTROL SYSTEM
1. Initialize an empty git repository (git init)
2. Stage all changes (git add .)
3. Create a commit with a message (git commit -m "YOUR MESSAGE HERE")

REPEAT THEY CYCLE OF
2. Stage all changes (git add .)

## Github / Gitbucket / Amazon / Google - CLOUD REPOSTIORY SERVICE (BACKING UP CODE)
1. Create a Repository on Github
2. Create a link between your LOCAL REPO and your REMOTE REPO (git remote add ORIGIN HTTP_URL)
3. Push your code to Github (git push ORIGIN BRANCH_NAME)
